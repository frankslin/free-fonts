# Black Sugar Plum Candy Webfonts

Chunked woff2 webfonts for Black Sugar Plum Candy, generated from lxgw/BlackSugarPlumCandy main@c9258372b2a0cef8a6bd49e2275336c27eb0bc82.

```html
<link rel="stylesheet" href="./black-sugar-plum-candy.css">
```

```css
body {
  font-family: 'Black Sugar Plum Candy', sans-serif;
  font-weight: 700;
}
```

The CSS uses 256-codepoint `unicode-range` chunks, so browsers only download the blocks needed by the page. The package includes the upstream bold TTF as a bold-weight webfont family.
