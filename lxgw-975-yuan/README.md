# LXGW 975 Yuan SC Webfonts

Chunked woff2 webfonts for LXGW 975 Yuan SC / 霞鹜975圆体SC, generated from lxgw/975Yuan release 26.06.20.

```html
<link rel="stylesheet" href="./lxgw-975-yuan.css">
```

```css
body {
  font-family: 'LXGW 975 Yuan SC', sans-serif;
}
```

The CSS uses 256-codepoint `unicode-range` chunks, so browsers only download the blocks needed by the page. The package includes weights 400 (Regular), 500 (Medium), and 700 (Bold).
