# LXGW WenKai GB Webfonts

Chunked woff2 webfonts for LXGW WenKai GB / 霞鹜文楷 GB, generated from lxgw/LxgwWenKaiGB v1.522.

```html
<link rel="stylesheet" href="./lxgw-wenkai-gb.css">
```

```css
body {
  font-family: 'LXGW WenKai GB', cursive, sans-serif;
}
```

The CSS uses 256-codepoint `unicode-range` chunks, so browsers only download the blocks needed by the page. The package includes the upstream Light, Regular, and Medium TTF faces as weights 300, 400, and 500.
