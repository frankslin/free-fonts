# Genki Serif / 源起明體 Webfonts

Chunked woff2 webfonts for `Genki Serif / 源起明體` TC, generated from ButTaiwan/genyo-font v2.100.

```html
<link rel="stylesheet" href="./genki-mincho.css">
```

```css
body {
  font-family: 'GenKiMin2TC', serif;
}
```

The CSS uses 256-codepoint `unicode-range` chunks, so browsers only download the blocks needed by the page. This package includes 7 weights: 200, 300, 400, 500, 600, 700, and 900.
