# WenJin Mincho Webfonts

Chunked woff2 webfonts for WenJin Mincho, generated from takushun-wu/WenJinMincho v2.012.

```html
<link rel="stylesheet" href="./wenjin-mincho.css">
```

```css
body {
  font-family:
    'WenJin Mincho Plane 0',
    'WenJin Mincho Plane 2',
    'WenJin Mincho Plane 3',
    serif;
}
```

The CSS uses 256-codepoint `unicode-range` chunks, so browsers only download the blocks needed by the page. The package includes the standard OTF P0/P2/P3 parts as regular-weight webfont families.
