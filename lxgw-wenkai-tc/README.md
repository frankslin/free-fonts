# LXGW WenKai TC Webfonts

Chunked woff2 webfonts for LXGW WenKai TC / 霞鶩文楷 TC, generated from lxgw/LxgwWenkaiTC v1.522.

```html
<link rel="stylesheet" href="./lxgw-wenkai-tc.css">
```

```css
body {
  font-family: 'LXGW WenKai TC', cursive, sans-serif;
}
```

The CSS uses 256-codepoint `unicode-range` chunks, so browsers only download the blocks needed by the page. The package includes the upstream Light, Regular, and Medium TTF faces as weights 300, 400, and 500.
